# 간격 · 반경 · 그림자 (Spacing · Radius · Elevation)

## Spacing (간격)

모든 여백은 **4px 배수** 스케일을 사용해요.

| 토큰 | 값 | 대표 용도 |
|---|---|---|
| `space.2` | 2px | 아이콘-텍스트 미세 간격 |
| `space.4` | 4px | 인접 요소 최소 간격 |
| `space.8` | 8px | 버튼 내부, 칩 간격 |
| `space.12` | 12px | 리스트 아이템 간격 |
| `space.16` | 16px | 카드 내부 패딩(기본) |
| `space.20` | 20px | 섹션 내부 패딩 |
| `space.24` | 24px | 화면 좌우 기본 여백 |
| `space.32` | 32px | 섹션 간 큰 여백 |
| `space.40` | 40px | 페이지 상하 여백 |

- 화면 좌우 기본 마진: **24px** (모바일), 콘텐츠 최대 폭 안에서 중앙 정렬.
- 리스트 아이템 세로 패딩: **16px**.

## Radius (모서리 반경)

| 토큰 | 값 | 용도 |
|---|---|---|
| `radius.sm` | 8px | 배지, 체크박스 |
| `radius.md` | 12px | 버튼, 입력 필드 |
| `radius.lg` | 14px | 큰 버튼, 인증 셀 |
| `radius.xl` | 16px | 카드 |
| `radius.2xl` | 22~24px | 다이얼로그, 바텀시트(상단) |
| `radius.full` | 9999px | 원형 아이콘 버튼, 스위치, 토스트 |

## Elevation (그림자)

떠 있는 정도를 그림자로 표현해요. 남발하지 말고 **레이어가 실제로 뜬 경우에만** 사용해요.

| 토큰 | 그림자 | 용도 |
|---|---|---|
| `elevation.1` | `0 1px 3px rgba(0,20,60,.12)` | 세그먼트 선택칩, 얕은 카드 |
| `elevation.2` | `0 2px 8px rgba(0,20,60,.12)` | 플로팅 버튼 |
| `elevation.3` | `0 8px 24px rgba(0,20,60,.24)` | 토스트, 바텀시트, 다이얼로그 |

## Border (구분선)

| 용도 | 값 |
|---|---|
| 기본 구분선 | `1px solid grey.200` (`#e5e8eb`) |
| 리스트 내부 얇은 선 | `1px solid grey.100` (`#f2f4f6`) |
| 입력 밑줄(기본) | `2px solid grey.200` |
| 입력 밑줄(포커스) | `2px solid blue.500` |

## 터치 타깃

누를 수 있는 모든 요소는 최소 **44×44px**의 히트 영역을 확보해요. 시각적으로 작아 보여도 패딩으로 영역을 넓혀요.
