# 컬러 (Colors)

색은 **의미 단위**로 사용해요. 값을 직접 하드코딩하지 말고 토큰 이름으로 참조하세요.

## Primary

| 역할 | 토큰 | HEX |
|---|---|---|
| Primary | `blue.500` | `#3182f6` |
| Primary (hover/pressed) | `blue.600` | `#2272eb` |
| Primary weak 배경 | `blue.50` | `#e8f3ff` |

## Grey Opacity

배경 위에 겹쳐 쓰는 반투명 회색이에요. 딤(dim), 구분선, 눌림 상태에 사용해요.

## 배경

| 토큰 | 값 | 용도 |
|---|---|---|
| `background` | `#FFFFFF` | 기본 배경 |
| `greyBackground` | `grey.100` (`#f2f4f6`) | 섹션 구분, 카드 밖 여백 |
| `layeredBackground` | `#FFFFFF` | 카드/시트 위 레이어 |
| `floatedBackground` | `#FFFFFF` | 떠 있는 요소(팝오버 등) |

## 의미별 사용 규칙

| 의미 | 텍스트/아이콘 | 배경 |
|---|---|---|
| 기본 텍스트 | `grey.900` | — |
| 보조 텍스트 | `grey.500`~`grey.600` | — |
| 비활성 | `grey.400` | `grey.100` |
| Primary 강조 | `blue.500` | `blue.50` |
| 성공 | `green.500` | `green.50` |
| 에러/위험 | `red.500` | `red.50` |
| 주의 | `orange.600` | `orange.50` |

**Do** — 지출은 `grey.900`(검정), 수입/강조는 `blue.500`으로 통일해요.
**Don't** — 같은 화면에서 강조색을 2개 이상 섞지 말아요.
