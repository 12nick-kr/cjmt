# Input — 입력 계열

값을 입력하거나 선택하는 컴포넌트들.

---

## TextField

밑줄형 단일 입력 필드.

**해부**: 라벨 · 입력 영역 · 밑줄 · (값이 있을 때) 삭제 버튼 · 도움말/에러 메시지.

| 상태 | 라벨 | 밑줄 |
|---|---|---|
| 기본 | `grey.500` | `2px grey.200` |
| 포커스 | `blue.500` | `2px blue.500` |
| 에러 | `red.500` | `2px red.500` + 하단 에러 메시지 |
| 비활성 | `grey.400` | `grey.200` |

- 값이 입력되면 오른쪽에 원형 삭제(✕) 버튼이 나타나요.
- 에러 메시지는 밑줄 아래 13px `red.500`.

### 변형
- **SplitTextField** — 인증번호처럼 자릿수를 칸으로 나눈 입력. 현재 입력 칸은 `blue.500` 테두리로 강조.
- **TextArea** — 여러 줄 입력. 우하단에 글자 수 카운터(선택).
- **SearchField** — 왼쪽 돋보기 아이콘 + 알약형(pill) 배경 `grey.100`. 입력 시 삭제 버튼.

---

## Checkbox

on/off 선택. 24×24px, `radius.sm`(8px).
- 선택: 배경 `blue.500` + 흰 체크. 미선택: 흰 배경 + `2px grey.300` 테두리.
- 약관 동의처럼 목록형은 "(필수)/(선택)" 접두어를 라벨에 붙여요.

## Switch

즉시 반영되는 on/off 토글. 저장 버튼 없이 켜자마자 적용될 때만 써요.
- 트랙 52×32px `radius.full`, 노브 28px 흰 원.
- ON: 트랙 `blue.500`, 노브 오른쪽. OFF: 트랙 `grey.300`, 노브 왼쪽. 전환 0.2s.

## Slider

범위 내 값 선택. 트랙 4px, 노브 24px 흰 원 + 그림자. 채워진 구간은 `blue.500`.
현재 값을 우측에 숫자로 함께 보여줘요.

## Stepper / Numeric Spinner

수량을 1씩 증감. `−` (중립, `grey.100`) · 숫자 · `+` (강조, `blue.500`).
최소값에서 `−`는 비활성 처리해요.

## Segmented Control

2~4개 중 하나를 고르는 세그먼트 필터. `grey.100` 트랙 안에서 선택된 칸만 흰 배경 + 얕은 그림자(`elevation.1`)로 떠 보여요.
탭(화면 전환)과 달리 **필터/모드 전환**에 써요.
