# Display — 정보 표시 계열

내용을 담아 보여주는 컴포넌트들.

---

## ListRow

목록의 기본 단위. **좌측 에셋 · 본문(제목/설명) · 우측 액세서리** 3영역으로 구성돼요.

- 좌측: 아이콘/썸네일/이니셜(44×44, `radius.lg`) — 없어도 됨.
- 본문: 제목(15px/600 `grey.900`) + 설명(13px `grey.500`).
- 우측: 금액·상태·스위치·chevron 등. 누르면 이동하는 행은 chevron(›)을 둬요.
- 세로 패딩 16px, 행 사이 `1px grey.100` 구분선. 누를 수 있으면 hover 시 `grey.50` 배경.

> 금액은 지출 `grey.900`, 수입/강조 `blue.500`으로 색을 통일해요.

## TableRow

좌우 **라벨 → 값** 형태의 정보 나열(주문 상세, 계좌 정보 등). 라벨은 `grey.500`, 값은 `grey.900`. 오른쪽 정렬 값.

## Board Row

공지/게시물 목록 행. 제목 + 날짜/작성자 메타. 안 읽음은 제목 앞 점(dot) 또는 Badge로 표시.

## Badge

작고 강한 상태 라벨. 12px/700, 패딩 5×10px, `radius.sm`.

| 의미 | 배경 / 텍스트 |
|---|---|
| 신규 | `blue.50` / `blue.500` |
| 완료·성공 | `green.50` / `green.500` |
| 지연·에러 | `red.50` / `red.500` |
| 대기·주의 | `orange.50` / `orange.600` |
| 카운트 | `grey.800` / 흰색 (원형) |

## Border

영역을 나누는 선/블록. 얇은 선은 `grey.100`~`grey.200`, 섹션을 크게 나눌 땐 8px 두께 `grey.100` 블록으로.

## Paragraph

본문 텍스트 블록. 17px/25.5 `grey.700`, 문단 사이 여백 16px. 긴 글의 가독성을 위한 기본 컨테이너.

## Highlight

문장 안 특정 단어 강조. `blue.500` 텍스트 또는 `yellow.100` 형광펜 배경 중 하나만 써요.

## Bubble

말풍선 형태의 짧은 안내/코치마크. 꼬리(tail) 방향으로 대상을 가리켜요. 배경 `grey.900` + 흰 텍스트(강조형) 또는 `blue.50`(정보형).

## Rating

별점 표시/입력. 채워진 별 `yellow.500`, 빈 별 `grey.200`. 0.5 단위 지원 가능.

## Chart — Bar Chart

막대 차트. 막대 `blue.500`(강조 시리즈) / `grey.200`(비교 시리즈). 축·격자선은 `grey.100`으로 최대한 옅게, 데이터에 집중시켜요.
