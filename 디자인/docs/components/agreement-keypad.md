# Agreement · Keypad · Asset

핀테크 특화 컴포넌트들.

---

## Agreement (약관 동의)

여러 약관을 한 번에 동의받는 묶음.
- 최상단 "전체 동의" 체크박스 → 하위 항목 전체 토글.
- 각 항목은 "(필수)/(선택)" 라벨 + 오른쪽 "보기 ›"로 상세 이동.
- 필수 항목이 모두 체크돼야 하단 CTA가 활성화돼요.
- 버전: **V3 / V4** — 신규 화면은 최신(V4) 사용을 권장.

## Keypad (보안 키패드)

민감한 숫자/문자 입력을 위한 자체 키패드. OS 키보드 대신 사용해 입력값을 보호해요.

| 종류 | 용도 |
|---|---|
| **Number Keypad** | 금액·전화번호 등 일반 숫자 |
| **Full Secure Keypad** | 비밀번호 — 키 배열이 매번 섞이고, 화면 캡처/로깅 방지 |
| **Alphabet Keypad** | 보안이 필요한 영문 입력 |

- 하단 고정, 키 높이 넉넉히(최소 44px), 삭제/완료 키 포함.
- 보안 키패드는 눌린 키를 화면에 그대로 노출하지 않아요(마스킹).

## Asset (에셋 / 아이콘·일러스트)

브랜드 일러스트·아이콘을 규격에 맞게 감싸 쓰는 래퍼.
- **Asset 이해하기** — 크기/여백 규격(frame) 안에서 사용, 임의 배율 금지.
- **Frame** — 표준 프레임에 넣어 정렬·여백을 자동으로 맞춰요.
- **래핑 컴포넌트** — 자주 쓰는 에셋은 이름 붙인 컴포넌트로 감싸 재사용.

> cjmt에서 실제 일러스트가 없을 땐 **줄무늬 플레이스홀더 + 설명 라벨**로 자리를 잡고, 실 에셋이 준비되면 교체해요. 임의로 SVG를 그려 넣지 않아요.
